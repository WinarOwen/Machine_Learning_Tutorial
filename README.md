## Installation
- Clone the repository
```git clone https://github.com/WinarOwen/Machine_Learning_Tutorial.git```
- Go to the folder
```cd Machine_Learning_Tutorial```
- Install dependencies
``` pip install -r requirements.txt```
- Experiment with the notebook!
